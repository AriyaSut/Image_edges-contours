# Edge&contours
Filter is modification of the pixel value that use more than 1 pixel to compute new pixel value.

## Overview

* Linear filter
  * Box filter
  * Gauss filter 
  * Laplace or Maxican hat filter
* Nonlinear filter
  * Minimum filter
  * Maximum filter
  * Median filter
  * Weight Median filter
